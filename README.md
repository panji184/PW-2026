# Sistem Pemesanan Tiket Pesawat

Sistem Pemesanan Tiket Pesawat adalah aplikasi berbasis web yang dibangun menggunakan Laravel untuk memudahkan pengguna dalam mencari, memesan, dan mengelola tiket penerbangan secara online.

Project ini dibuat sebagai implementasi konsep CRUD (Create, Read, Update, Delete) dan autentikasi pengguna dalam framework Laravel.
